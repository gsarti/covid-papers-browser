Interrogate the Covid-19 Semantic Browser API as follows:

  • /paper?<query=QUERY>[&count=COUNT] : Returns top COUNT papers relevant to QUERY, default top 10.
  • /paper/<ID>[?query=QUERY&count=COUNT] : Returns ID's paper info, plus its top COUNT (default 10) relevant paragraphs if QUERY is specified.
  • / or /help : Prints this usage guide

<PARAM> denotes that PARAM is mandatory, [PARAM] that it is optional.
