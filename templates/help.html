Interrogate the Covid-19 Semantic Browser API as follows:

  • /years[?count=COUNT] : Returns unique count of papers for each year
  • /authors[?count=COUNT] : Returns unique count of papers for each author
  • /journals[?count=COUNT] : Returns unique count of papers for each journal
  • /licenses[?count=COUNT] : Returns unique count of papers for each licenses
  • /paper?<query=QUERY>[&count=COUNT][&score=SCORE][&author=AUTHOR][&year=YEAR][&journal=JOURNAL][&license=LICENSE] : Returns top COUNT papers relevant to QUERY by at least SCORE, default top 10.
  • /paper/<ID>[?query=QUERY][&count=COUNT][&score=SCORE] : Returns ID's paper info, plus its top COUNT (default 10) relevant paragraphs by at least SCORE if QUERY is specified.
  • / or /help : Prints this usage guide

<PARAM> denotes that PARAM is mandatory, [PARAM] that it is optional.
